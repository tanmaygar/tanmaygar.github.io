Fluttered With Flutter

Wanna learn how to build native cross platform apps? Sign up for our Flutter workshop series! "Fluttered with Flutter" is an introductory Flutter workshop series organized by DSC@MMU, with 3 sessions. We've invited two Google Developer Experts on Flutter:

  • Muhammad Haris Bin Samingan, a Flutter developer and YouTuber in Singapore (P/S: He teaches Flutter on his YouTube channel, go check it out)
  • Joshua de Guzman, a software engineer at Better Bank - Brooklyn, New York
This workshop series is beginner friendly, as we'll go from the fundamentals of programming with Dart, to having a basic understanding of Flutter.

Full Stack Web Development

Wanna learn full stack website development, from frontend to backend? Sign up for this workshop series!

Here are the pre-requisites for this workshop series:

  • basic understanding of HTML, CSS
  • JavaScript (variables, loops, functions, classes, JSON notation)
  • fundamental modern JavaScript (ES6)
  • basic understanding of the client-server model
Don't worry if you don't have a good grasp of the pre-requisites, after watching the following two videos, you'll be ready for the workshop series.
  • Video 1 : Git, HTML, CSS
  • Video 2 : JavaScript Basics
