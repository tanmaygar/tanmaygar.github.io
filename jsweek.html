DSC@MMU is announcing Javascript Week 2021!

React.js, Node.js, PostgreSQL, Vanilla JS

12 July 2021 - 16 July 2021

What You'll Learn

This is an entire week dedicated to exploring the Javascript stack via workshops and hands-on experience.

React.js

React is a rich frontend library used by Facebook, Canva and Twitter.

Node.js

Node is a powerful backend runtime engine that operates on a single event loop.

PostgreSQL

An incredibly functional relational database built to scale to the moon.

And yes, a final project that'll help apply what you've learnt. This is a bootcamp at absolutely no cost.

Schedule

  • Day 1- Introduction to Javascript (12th July)
  • Day 2- React.js (13th July)
  • Day 3- Node.js (Express) (14th July)
  • Day 4- Introduction to SQL and PostgreSQL (15th July)
  • Day 5- Build a CRUD app using React, Node and Postgres (16th July)

Requirements

Software you'll need
  • Visual Studio Code
  • Beekeeper
  • Node
Hardware you'll need
  • A Computer with an internet connection
We'll update this page with pre-requisites and installation videos nearer to the event!
